# vessel-management-tool

### Mongo DB Instalation

Install Brew
```bash
ruby -e "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install)"
```
Update Repository
```bash
brew update
```
Install Mongoo DB
```bash
brew install mongodb
```
Run the database in background
```bash
sudo mongod &
```

### Run Server

Run Server:
```bash
node server.js
```
